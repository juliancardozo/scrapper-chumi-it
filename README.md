# Scrapper Chumi IT

This repository contains a sample **n8n** workflow that scrapes job listings from multiple pages of a website. The workflow can be imported directly into n8n.

## Files

- `workflow-example.json` – Example workflow export for n8n. It fetches three pages, extracts job titles, company names, and links using CSS selectors, and is triggered by a daily Cron node.

To use the workflow, open n8n, choose **Import workflow**, and select the JSON file.
